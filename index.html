I'm a div

Pointers for Div

  1. Width of a block level element is 100% of the viewport width
  2. Height of block isn't set by default
  3. Height & Width can be explicitly set using CSS
I'm a span

Pointers for In-line

  1. Width of a Inline level element is dependent on the content
  2. Height of In-line isn't set by defaul
  3. Height & Width can not be explicitly set using CSS
